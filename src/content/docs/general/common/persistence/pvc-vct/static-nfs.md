## Appears in

- `.Values.persistence.$name.static`

:::tip

- See available persistence keys [here](./index.md).
- This options apply only when `type: pvc` or `type: vct` and `mode: nfs`.

:::

---

## `server`

Define the nfs server

|            |                                   |
| ---------- | --------------------------------- |
| Key        | `persistence.$name.static.server` |
| Type       | `string`                          |
| Required   | ✅                                |
| Helm `tpl` | ✅                                |
| Default    | `""`                              |

Example

```yaml
persistence:
  nfs-vol:
    type: pvc
    static:
      server: /server
```

---
