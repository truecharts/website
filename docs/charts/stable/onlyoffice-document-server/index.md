# onlyoffice-document-server

![Version: 9.0.18](https://img.shields.io/badge/Version-9.0.18-informational?style=flat-square) ![Type: application](https://img.shields.io/badge/Type-application-informational?style=flat-square) ![AppVersion: "7.2.2"](https://img.shields.io/badge/AppVersion-"7.2.2"-informational?style=flat-square)

ONLYOFFICE Document Server is an online office suite comprising viewers and editors for texts, spreadsheets and presentations, fully compatible with Office Open XML formats: .docx, .xlsx, .pptx and enabling collaborative editing in real time.
