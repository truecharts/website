# Input Validation

**`Speed`**
Accepted formats are:

- Integers (eg `1`, `2`)
- Floats (eg `1.25`, `1.50`, `2.5`)

Regex used to match those: `^[0-9]+(\.[0-9]{1,2})?$`
You can try live [here](https://regex101.com/r/4HI4wW/1)
