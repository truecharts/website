# Validación de entrada

__`LATITUDE`__ y __`LONGITUDE`__ Formatos aceptados son:

- Float (eg. `24.2028`)

Regex utilizada para emparejar esto: `^[0-9]{1,}\.{1}[0-9]{1,}$` Puedes probar en vivo [aquí](https://regex101.com/r/xsLGWN/1)
