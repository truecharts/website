# 默认 Helm-value

TrueCharts 主要是为了提供 TrueNAS SCALE 应用而构建的。 然而，我们也提供所有应用，作为标准的帮助图。 在这份文件中，我们的目标是在我们的共同图表中记录默认值。 这张图表被我们的许多应用用于提供智能默认和逻辑。
