# Validation de l'entrée

__`LATITUDE`__ et __`LONGITUDE`__ Formats acceptés sont :

- Float (eg. `24.2028`)

Regex a utilisé cette correspondance : `^[0-9]{1,}\.{1}[0-9]{1,}$` Vous pouvez essayer de vivre [ici](https://regex101.com/r/xsLGWN/1)
