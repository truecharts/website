# Eingabeüberprüfung

__`LATITUDE`__ und __`LONGITUDE`__ Akzeptierte Formate sind:

- Float (eg. `24.2028`)

Regex passt zu diesem: `^[0-9]{1,}\.{1}[0-9]{1,}$` Du kannst live versuchen [hier](https://regex101.com/r/xsLGWN/1)
